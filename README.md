🚖 Uber Ride Analytics Dashboard | Power BI Project
Developed an interactive Uber Ride Analytics Dashboard using Power BI to analyze ride performance, revenue trends, customer behavior, and operational KPIs.

🔍 Key Insights:
📊 93K+ Completed Bookings Analysis
💰 51.85M Total Revenue Tracking
📈 Monthly Revenue & Booking Trend Analysis
🚗 Revenue by Vehicle Type (Auto, Go Mini, Sedan, Bike, XL)
⭐ Customer & Driver Rating Performance Monitoring
📍 Top Pickup & Drop Location Insights
🛠 Tools & Skills Used:

Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Modeling
- KPI Dashboard Design
- Business Intelligence & Data Visualization

Dataset referenced and structured based on a YouTube tutorial for educational and practice purposes.
